Clínica Médica – CRUD Completo

Projeto desenvolvido em sala de aula – SENAI São Carlos
Tecnologia utilizada: JavaScript (Vanilla)

Descrição do Projeto

Este projeto consiste no desenvolvimento de um sistema CRUD completo para gerenciamento de pacientes de uma clínica médica. O objetivo foi aplicar os conceitos fundamentais de manipulação de dados no front-end utilizando JavaScript puro, sem frameworks ou bibliotecas externas.

O sistema permite cadastrar, visualizar, atualizar e excluir registros, utilizando recursos de armazenamento local e manipulação dinâmica do DOM.

Funcionalidades

Cadastro de pacientes

Listagem de registros em tabela dinâmica

Edição de informações diretamente a partir da listagem

Exclusão de registros

Validação básica dos campos do formulário

Armazenamento dos dados no LocalStorage

Interface simples e funcional para fins didáticos

Como Executar o Projeto

Baixe ou clone este repositório.

Abra o arquivo index.html diretamente no navegador.

Todas as funcionalidades estarão disponíveis localmente, sem necessidade de servidor externo ou instalação adicional.

Objetivo Educacional

O projeto foi criado como exercício prático durante as aulas do SENAI São Carlos, com foco em reforçar os fundamentos do desenvolvimento web e demonstrar como é possível construir aplicações funcionais utilizando apenas JavaScript puro.
